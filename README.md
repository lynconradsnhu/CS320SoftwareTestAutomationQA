# CS320SoftwareTestAutomationQA

1. How can I ensure that my code, program, or software is functional and secure?

Run frequent builds and tests, include white box testing via unit tests and black box testing where possible. Adhere to security practices and perform security and integration testing where needed. 

2. How do I interpret user needs and incorporate them into a program?

Given requirements and user stories. Code should be written according to these specifications with matching tests.

3. How do I approach designing software?

Understand the requirements and user stories and develop using best practices keeping testing in mind and part of the development process.
